Hoş Geldiniz

Bir takım gereksinimleri karşılamak için yazılmış bir HTTP server'ıdır.

Endpoint'ler

  • /api/hello - JSON formatında bir yanıt döner
  • /static/style.css - CSS dosyasını döner
  • /api/data - Örnek resim gösterir

Örnek Resim

Özellikler

  • GET isteklerini işler
  • Statik dosyaları (HTML, CSS, JS) sunar
  • API endpointleri aracılığıyla JSON yanıtları sağlar
  • MIME türü işleme
  • Çoklu bağlantı işleme
  • Temel hata işleme (404, 500)
